개인정보 보호정책

최종 수정: 2026년 3월 24일

버터떡맵(이하 "서비스")은 이용자의 개인정보를 소중하게 생각합니다. 본 개인정보 보호정책은 서비스 이용 시 수집되는 개인정보와 그 처리에 관한 내용을 설명합니다.

1. 수집하는 개인정보

서비스는 다음과 같은 개인정보를 수집합니다:

1.1 필수 수집 항목

수집 항목 설명 수집 방법
위치 정보 현재 위치 기반 근처 매장 검색 브라우저 위치 권한
쿠폰 클레임 기록 사용자가 클레임한 쿠폰 정보 로컬스토리지에 저장

1.2 선택 수집 항목 (사장님 로그인)

사장님 대시보드 이용 시 소셜 로그인을 통해 다음 정보가 수집됩니다:

  • Google 로그인: 이메일, 이름, 프로필 사진
  • Kakao 로그인: 이메일, 닉네임, 프로필 사진
  • Naver 로그인: 이메일, 이름, 프로필 사진
  • 재고 정보: 각 매장의 상품 재고 데이터

2. 개인정보 수집 목적

  • 위치 정보: 사용자의 현재 위치에서 가까운 버터떡 매장 검색 기능 제공
  • 쿠폰 데이터: 이용자가 클레임한 쿠폰 기록 보관 및 중복 클레임 방지
  • 사장님 정보: 재고 관리 대시보드 접근 권한 확인 및 계정 관리
  • 재고 정보: 실시간 매장 상품 재고 상태 관리 및 조회

3. 개인정보 보유 및 이용 기간

정보 종류 보유 기간 설명
쿠폰 클레임 기록 사용자 브라우저 로컬스토리지 브라우저 쿠키/캐시 삭제 시 제거
사장님 로그인 정보 계정 삭제 시까지 Firebase에서 관리
매장 재고 데이터 최신 데이터 유지 Supabase에서 관리

4. 제3자 정보 제공

서비스는 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우 예외가 적용됩니다:

  • 법령에 따른 요청: 법적 의무에 따른 경우
  • 클라우드 서비스 제공자: 서비스 운영을 위해 다음 업체와 협력합니다:
    • Firebase: 사장님 계정 인증 및 관리 (Google 제공)
    • Supabase: 매장 및 재고 데이터 저장소
    • Naver Maps API: 지도 및 위치 서비스
    • Kakao JavaScript Key: 소셜 로그인

5. 정보보안

서비스는 사용자의 개인정보 보호를 위해 다음과 같은 조치를 취합니다:

  • 위치 정보는 브라우저에서만 처리되며, 서버에 저장되지 않습니다
  • 쿠폰 클레임 데이터는 로컬스토리지에만 저장되어 개인 기기에만 존재합니다
  • 사장님 계정은 Firebase의 보안 인증 프로토콜을 통해 보호됩니다
  • HTTPS 암호화 연결을 사용합니다

6. 이용자의 권리

사용자는 다음의 권리를 가집니다:

  • 접근권: 본인 개인정보 조회 요청
  • 수정/삭제권: 개인정보 수정 및 삭제 요청
  • 처리 정지권: 개인정보 처리 중지 요청
  • 위치 정보 거부권: 브라우저 설정에서 위치 권한 거부 가능
  • 쿠폰 데이터 삭제: 브라우저 로컬스토리지 직접 삭제 가능
참고: 위치 정보는 서비스 서버에 저장되지 않으므로, 별도의 삭제 요청이 필요 없습니다. 각 브라우저의 위치 권한 설정을 통해 언제든지 제어할 수 있습니다.
